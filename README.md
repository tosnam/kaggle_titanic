# kaggle_titanic

[Kaggle competition site](https://www.kaggle.com/competitions/titanic)

Kaggle Titanic data를 사용하여 machine learning과 data science 기초를 공부한 코드를 업로드한다.

각 python 파일에 노트북 작성에 참고한 커널 정보를 기록해뒀다.
